COVID-19 Vaccination Dashboard

By: Mason Totall, Daniel Bernal, Steven Gaetz & Rebecca Tricker

Since January 2020, Covid-19 has spread world-wide infecting millions. For many, the release of the Covid-19 vaccine suggested an end in sight to the long lasting 2020 pandemic. The following dashboard analyzes Covid-19 vaccinations in the United States. Each state has been analyzed so that the below visualizations illustrate the percentage of population vaccinated, number of people vaccinated in the United States and per state, and number of Covid-19 cases per state. Each visualization analyzes 111 days worth of vaccine data from January 14, 2021 to May 3, 2021.

Percentage of Population Vaccinated
Number of Vaccines Initiated & Completed
Vaccines Initiated: This indicates the number of people vaccinated with the first dose
Vaccines Completed: This indicates the number of people vaccinated with both the first and second dose
Infection Rate
Infection Rate R(t): The estimated number of new people each COVID-positive person will infect.
R(t) greater than 1.0: Daily new Covid-19 cases are increasing
R(t) less than 0.9: Covid-19 cases are decreasing
